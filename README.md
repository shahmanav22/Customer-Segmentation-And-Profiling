# Customer Segmentation & Profiling using K-Means Clustering

## Overview

This capstone project applies K-Means Clustering to segment customers based on demographic and purchasing behavior. The goal is to identify meaningful customer groups and provide actionable marketing strategies for each segment.

## Objectives

- Analyze customer behavior
- Perform customer segmentation
- Profile customer groups
- Recommend targeted marketing strategies
- Support business decision-making

## Dataset

- Mall Customers Dataset (Kaggle)

## Workflow

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Feature Scaling
- Elbow Method
- K-Means Clustering
- Silhouette Score Evaluation
- Customer Profiling
- Business Recommendations

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- SciPy
- Joblib

## Key Findings

- Five distinct customer segments were identified.
- Customer behavior varies significantly across age, income, and spending.
- Personalized marketing can improve customer engagement and retention.
- Clustering provides valuable insights for strategic business decisions.
